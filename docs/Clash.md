# Clash for linux

```bash
git clone git@github.com:wnlen/clash-for-linux.git
```

## 修改 `.env` 配置文件

```bash
# Clash 订阅地址
export CLASH_URL='https://feedneo.com/files/******/clash.yml'
export CLASH_SECRET=''
```
